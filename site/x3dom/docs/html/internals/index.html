X3DOM internals¶

Documentation for people who are hacking on X3DOM itself. This is the place to go if you’d like to help improve X3DOM or learn about how X3DOM works “under the hood”.

Warning

Elsewhere in the X3DOM documentation, coverage of a feature is a sort of a contract: once an API is in the official documentation, we consider it “stable” and don’t change it without a good reason. APIs covered here, however, are considered “internal-only”: we reserve the right to change these internals if we must.
